# OpenDeepWiki

OpenDeepWiki 是一个开源的、AI 驱动的代码知识库。它分析来自 GitHub、GitLab、Gitee、Gitea 等平台的代码仓库，然后自动生成结构化文档、架构图和知识图谱 — 一切只需几分钟。

基于 **.NET 9** 和 **Semantic Kernel**，配合 **Next.js** 前端，OpenDeepWiki 架起了原始代码和人类可读文档之间的桥梁。

## 工作原理

1. **克隆** — OpenDeepWiki 在本地克隆目标仓库
2. **扫描** — 递归扫描目录，尊重 `.gitignore`，可选择使用 AI 过滤大型仓库
3. **分析** — AI 生成目录结构、项目概览和每个模块的详细文档
4. **服务** — 文档存储在数据库中，通过 Next.js 前端提供服务，支持全文搜索、MCP 和对话式 AI
