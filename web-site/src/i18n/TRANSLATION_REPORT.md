## 翻译改进详情

### 🇬🇧 英文 (en-US) 改进

#### 核心改进原则
1. **简洁明了** - 去除冗余，使用更简洁的表达
2. **技术术语统一** - 确保全文术语一致
3. **自然英文表达** - 避免直译，使用地道表达
4. **用户体验优化** - 友好的提示语言

#### 主要改进示例

**通用部分 (Common)**
| 原翻译 | 改进后 | 说明 |
|--------|--------|------|
| `Submitting...` | `Please wait...` | 更友好的加载提示 |
| `Next Page` | `Next` | 简洁化 |
| `Previous Page` | `Previous` | 简洁化 |
| `{{count}} minutes ago` | `{{count}} min ago` | 简短时间格式 |

**主页部分 (Home)**
| 原翻译 | 改进后 | 说明 |
|--------|--------|------|
| `No repository data available` | `No repositories found` | 更自然的空状态提示 |

**仓库部分 (Repository)**
| 原翻译 | 改进后 | 说明 |
|--------|--------|------|
| `Mind Map` | `Knowledge Graph` | 技术术语准确性 |
| `Loading mind map...` | `Loading knowledge graph...` | 术语一致性 |
| `Mind map loading failed` | `Knowledge graph loading failed` | 术语一致性 |

**管理后台 (Admin)**
| 原翻译 | 改进后 | 说明 |
|--------|--------|------|
| `Previous Page` | `Previous` | 简洁化 |

## 文件结构

### 目录组织
```
web-site/src/i18n/locales/
├── en-US.json              # 英文主翻译文件
├── zh-CN.json              # 中文主翻译文件
├── ja-JP.json              # 日文主翻译文件
├── ko-KR.json              # 韩文主翻译文件
├── admin/
│   ├── en-US.json         # 英文管理后台
│   ├── zh-CN.json         # 中文管理后台
│   ├── ja-JP.json         # 日文管理后台
│   └── ko-KR.json         # 韩文管理后台
├── i18n-complete-supplement-*.json  # 补充翻译文件
└── en-US-admin-supplement.json      # Admin补充文件
```

### 主翻译文件结构
每个主翻译文件包含以下顶级键：
- `common` - 通用UI元素（49个条目）
- `nav` - 导航菜单（10个条目）
- `home` - 主页内容（9个条目）
- `footer` - 页脚信息（4个条目）
- `login` - 登录页面（9个条目）
- `register` - 注册页面（9个条目）
- `repository` - 仓库管理（6个条目）
- `messages` - 系统消息（5个条目）
- `admin` - 管理后台（5个条目）
- `settings` - 系统设置（47个条目）
- `profile` - 用户资料（3个条目）

### Admin命名空间结构
- `title` - 页面标题
- `messages` - 权限和加载消息
- `common` - 共通UI元素
- `nav` - 管理导航
- `dashboard` - 仪表板（71个条目）
- `users` - 用户管理（41个条目）
- `repositories` - 仓库管理（30-31个条目）
- `roles` - 角色管理（39个条目）
- `tokens` - Token使用统计（5个条目）
